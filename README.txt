La Casa De Las Iguanas

Un juego de simulación y estrategia de colonias.

🚀 Cómo Jugar

Este juego requiere Python 3 y algunas librerías para funcionar.

1. Descarga el Proyecto
Haz clic en el botón verde "<> Code" arriba y selecciona "Download ZIP". Descomprime el archivo.

⚠️ NOTA IMPORTANTE (FALSO POSITIVO)

Si tu antivirus detecta una amenaza al descargar el ZIP o al ejecutar el archivo compilado (.exe), ignórala. Esto se debe a que la herramienta de compilación de Python (PyInstaller) utiliza un formato de archivo que a menudo es marcado erróneamente por los escáneres de virus como sospechoso. El código es seguro.

2. Instala las Dependencias
Abre una terminal (CMD, PowerShell, Terminal) en la carpeta del proyecto que acabas de descomprimir y ejecuta:

pip install -r requirements.txt


3. ¡Juega!
Una vez instaladas las dependencias, ejecuta el juego con:

python -m src.main
