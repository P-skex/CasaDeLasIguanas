Un juego de simulación y estrategia de colonias.

🚀 Cómo Jugar

Este juego requiere Python 3 y algunas librerías para funcionar.

1. Descarga el Proyecto
Haz clic en el botón verde "<> Code" arriba y selecciona "Download ZIP". Descomprime el archivo.

2. Instala las Dependencias
Abre una terminal (CMD, PowerShell, Terminal) en la carpeta del proyecto que acabas de descomprimir y ejecuta:

pip install -r requirements.txt


3. ¡Juega!
Una vez instaladas las dependencias, ejecuta el juego con:

python -m src.main


🎮 Controles

Controles Generales

[Flechas]: Mover la Cámara

[Clic Izquierdo]: Seleccionar Iguana (solo aliadas)

[ESC]: Deseleccionar / Cerrar Menú / Salir

Controles de Tribu (Sin selección)

[1, 2, 3]: Recolectar (Comida, Madera, Piedra)

[F]: Ataque General (Guerreros)

[C]: Curar Enfermedad (Curanderos)

[Z]: Abrir Centro de Acopio

[T]: Abrir Estado de la Tribu

[F1-F4]: Usar Artefactos

Controles de Iguana Seleccionada

[WASD]: Mover Iguana

[H]: Comer (Cura 10 V, consume 5 Comida)

Controles de Debug

[0]: Prueba de Estrés (Genera 50 guerreros)

[O]: Iniciar Lluvia Ácida

[8]: Iniciar Suelo es Lava
